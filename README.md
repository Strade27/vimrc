## <a name="usage">Usage

```sh
$ cd ~
$ git clone https://github.com/strade27/vimrc ~/.vim
$ ln -s .vim/vimrc .vimrc
# If you use NeoVIM
$ ln -s ~/.vim .config/nvim
$ vim +PlugInstall +qall now
```
